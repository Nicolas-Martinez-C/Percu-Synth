GENERAR FIRMWARE

Mismo motor de síntesis que esta webapp → el hardware suena igual.

Descarga arp_matrix_percusynth.ino y flashéalo desde el Arduino IDE.

Board ESP32S3 Dev Module · USB CDC On Boot Enabled · Flash Mode DIO (OPI rompe el I2S) · PSRAM OPI PSRAM. Salida por el DAC PCM5102.

El firmware corre el arpegiador y la síntesis de forma autónoma con sus pots/botones/IMU, y transmite p0..p3,b1..b5,imuX,imuY a 50 Hz para que la webapp lo espeje.
